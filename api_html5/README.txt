Le principe de Drag & Drop est assez simple il permet de glisser un élément d’une page web fonctionnelle (dossier, fichie..) tout en restant appuie sur la souris puis le déposer dans un autre endroit.
A la base les éléments d’une interface web ne sont pas déplaçables, ceci est possible grâce ce concept d'un simple déplacement de souris.
Il y a trois étapes : Glisser, rester appuie, déposer
